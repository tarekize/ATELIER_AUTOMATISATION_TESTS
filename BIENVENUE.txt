Étape 1 : Configuration (1 min)
   $ cp .env.example .env
   → Éditer .env et ajouter : IPSTACK_API_KEY=votre_clé

Étape 2 : Installation (1 min)
   $ pip install -r requirements.txt

Étape 3 : Validation (1 min)
   $ python quickstart_run.py
   OU
   $ python verify_setup.py

Étape 4 : Lancement (1 min)
   $ python flask_app.py
   → Ouvrir http://localhost:5000/dashboard

Étape 5 : Test (1 min)
   → Cliquer "Exécuter tests"
   → Voir résultats en live

      → Copier clé
      → Ajouter dans .env

   ⏱️  Rate Limiting
      → IPStack gratuit = 10 requêtes/minute
      → Ne pas exécuter tests trop souvent
      → Minimum 5-10 minutes entre runs

   🛡️  Sécurité
      → JAMAIS committer .env
      → JAMAIS hardcoder clé API
      → TOUJOURS utiliser secrets GitHub
      → .gitignore vous protège
