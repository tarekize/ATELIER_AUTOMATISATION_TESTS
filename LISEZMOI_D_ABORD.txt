    python test_config.py

💡 TIPS IMPORTANTS
==================

1. NE PAS COMMITTER .env !!
   (C'est dans .gitignore, rest cool 😎)

2. IPStack gratuit = 10 req/min MAX
   → Ne pas exécuter tests plus souvent

3. La clé API dans SECRETS GitHub, pas en code

4. Si erreur 429 : augmenter délai tâche planifiée

5. Logs disponibles via /error.log sur PythonAnywhere


📞 BESOIN D'AIDE ?
==================

Erreur "ModuleNotFoundError: tester" ?
   → Vérifier tester/__init__.py existe

Erreur "IPSTACK_API_KEY not found" ?
   → Configurer .env ou variable d'environnement

Erreur "Connection refused" ?
   → Vérifier clé API et connectivité Internet

Erreur "429 Too Many Requests" ?
   → IPStack limité 10 req/min, augmenter délai

Autres problèmes ?
   → Consulter DEPLOYMENT.md section Troubleshooting
   → Vérifier /error.log PythonAnywhere
